# Public Demo Post — Readiness Checklist

Judges evaluate the **public demo post**, and Devpost says it should be
self-explanatory. An empty post reads as a dead game. Use this checklist to make
the post feel alive before submission.

## 1. Create the post + seed it

1. After the private smoke passes, `npm run launch` (or `npm run dev` for a playtest sub) and install on your
   demo subreddit.
2. Run the mod menu action **"One More Dawn: seed demo state"** on the post.
   This loads a Day-5 city mid-raid: 9 citizens, The Marked mid-rescue, live
   crisis + council votes, faction standings, and a 3-day chronicle.
3. Open the post yourself and confirm the hook reads in ~10 seconds.

## 2. Drive real activity (small tester pass)

Get 5–10 friends/alts to open the post and each do a few things, so the tallies
and feed reflect real people, not just the seed:

- [ ] pledge to save The Marked
- [ ] cast the crisis vote
- [ ] back a council plan
- [ ] take a city action
- [ ] leave one real Reddit strategy comment through the City Chatter Hub
- [ ] return after a force-resolve / the next dawn to see the Dawn Report

Use **"One More Dawn: force-resolve day"** so you can show a real dawn transition
(city summary + your impact + a vital changing) without waiting 24h. Mention in
the post description that the city is seeded to a mid-run survival state.
